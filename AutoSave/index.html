Como funciona:

  • O conteúdo é salvo automaticamente no LocalStorage do seu navegador a cada vez que você digita
  • O contador de caracteres é atualizado em tempo real
  • Quando você recarrega a página, o texto é recuperado automaticamente
  • Use o botão "Limpar rascunho" para remover o conteúdo salvo
  • Os dados ficam armazenados apenas no seu computador

Os dados são salvos apenas no seu navegador usando localStorage e não são enviados para servidores externos.
